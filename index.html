Welcome to Swagger UI Themes.

One of the goals of Swagger UI Themes is to produce high-quality themes that you can use to spice up how your Swagger docs look.

How to install

  1. Download the zip
  2. Select one of the stylesheets in the themes folder
  3. Add stylesheet to the source of swagger-ui theme html page
  4. Make sure to either remove/comment out the link to screen.css or load the desired theme after it to override the default Swagger UI styles.
